Esta acción actualiza los datos del usuario tomándolos desde su plataforma de origen (por ejemplo, Facebook o WhatsApp) — útil cuando necesitas refrescar la información de perfil de alguien.

Al ejecutarla, se trae la **foto de perfil más reciente** del usuario y se actualiza su **zona horaria**.

<Card type="warning">
  Esta actualización solo funciona dentro de una ventana de **24 horas** desde la última interacción del usuario.
</Card>
