Botky te permite ejecutar lógica de flujo compleja usando código **JavaScript** directamente dentro de un nodo de Acción.

## Cómo acceder

Crea un bloque de nodo de Acción, entra a **Acciones avanzadas**, y selecciona el nodo **Función de JavaScript**.

## Reglas importantes

<Card type="warning">
  Después de escribir tu código, **siempre debes hacer un `return`** de tu valor — el bloque de función actúa como una función normal de JavaScript.
</Card>

<Card type="warning">
  Los métodos nativos de JavaScript que requieren un navegador (como `console.log`) **no funcionan** dentro del nodo de Función de JavaScript.
</Card>

<Card type="note">
  Al trabajar con JSON, primero debes **parsear** el JSON antes de poder usarlo dentro de tu código.
</Card>

El bloque de función soporta múltiples librerías de JavaScript que extienden sus capacidades nativas.
